# ADR 0004: Visual Peel-The-Layers Experience

## Status

Accepted.

## Context

The product must communicate to CEOs, CTOs, and engineers at the same time.

## Decision

Every workflow run must expose three levels of depth:

- Executive outcome.
- Architecture view.
- Engineering code/trace/policy view.

Every node in the graph must be inspectable.

## Consequences

The UI is not decorative. It is the primary explanation mechanism for the stack depth.
